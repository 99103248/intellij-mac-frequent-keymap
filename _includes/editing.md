### 编辑（Editing）

| Operation | Mac OSX 10.5+ |
| :---      | :---        |
| 基本代码补全 | ^+Space |
| 智能补全 | ^+⇧+Space |
| 语句补全 | ⇧+⌘+Enter |
|||
| 从上方开始一行 | ⌥+⌘+Enter |
| 从下方开始一行 | ⇧+Enter |
| 上移/下移一行 | ⇧+⌥+↑/↓ |
| 上移/下移一个块 | ⇧+⌘+↑/↓ |
| 选中一个代码块 | ⌥+↑ |
| 取消代码块选中 | ⌥+↓ |
|||
| 重复当前行 | ⌘+D |
| 复制并且不删除当前行 | ⌘+C |
| 剪切当前行并复制到粘贴板 | ⌘+X |
| 关闭当前标签 | ⌘+W |
| 注释当前行 | ⌘+/ |
|||
| 优化import | ^+⌥+O |
| 格式化代码 | ⌥+⌘+L |
